!!! note
    This is an independently maintained fork of
    [SCRT-HQ/PSGSuite](https://github.com/SCRT-HQ/PSGSuite), originally created
    by Nate Ferrell (SCRT HQ). The upstream project is no longer actively
    maintained; this fork exists to keep the module working. PSGSuite is
    distributed under the Apache 2.0 license with attribution retained in the
    repo's [NOTICE](https://github.com/jboiselle/PSGSuite/blob/main/NOTICE) file.

## About

PSGSuite is a Powershell module for Google/Google Workspace API calls wrapped in handy functions. Authentication can be established using a service account via a P12 key to negate the consent popup and allow for a greater hands off automation capabilities.
